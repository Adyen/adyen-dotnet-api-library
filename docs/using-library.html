Using the library

Set up a client and set the credentials to the Adyen platform. 

          
var client = new  Client("Company.PluginDemo", "password", Environment.Test, "ApplicationName")
{
  Config =
  {
    HmacKey = "8D153DA90DC126F9F1EF84E9CD67C081EDA8984439EADASDFGHJKLA123456789",
    SkinCode = "qz7R381z",
  }
};

The example connects you to the Adyen test platform. If you want to use our live platform use Environment.Live: 

var client = new Client ("Company.PluginDemo","password",
Environment.Test,"ApplicationName");

You can now use the following services:

  • Payment
  • Recurring
  • Modification
  • Directory Lookup
