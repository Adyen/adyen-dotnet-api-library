Using the library

Set up a client and set the credentials to the Adyen platform. 

          var client = new Client("WebserverUsername", "WebserverPassword", Environment.Test, "YourApplicationName"); 
        
$client = new \Adyen\Client();
$client->setApplicationName("Adyen PHP Api Library Example");
$client->setUsername("YOUR USERNAME");
$client->setPassword("YOUR PASSWORD");
$client->setEnvironment(\Adyen\Environment::TEST);

The example connects you to the Adyen test platform. If you want to use our live platform use: 

$client->setEnvironment(\Adyen\Environment::LIVE);

You can now use the following services:

  • Payment
  • Recurring
  • Modification
  • Directory Lookup

By default the input and output parameters are of type array. If you would like to use JSON you can doing the following thing:

$client->setInputType('json');
$client->setOutputType('json');
