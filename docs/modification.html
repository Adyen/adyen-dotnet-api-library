Modification

The payment service has the following methods:

capture

Using this you can capture the payment. 

var client = new Client();
var modification = new Modification(client);

var captureRequest = new CaptureRequest
{
  MerchantAccount = "MerchantAccount",
  ModificationAmount = new Amount("EUR", 150),
  Reference = "Merchant reference ",
  OriginalReference = pspReference
};

var captureResult = modification.Capture(captureRequest);

cancel

var client = new Client();
var modification = new Modification(client);

var captureRequest = new CancelRequest
{
  MerchantAccount = "MerchantAccount",
  Reference = "Merchant reference ",
  OriginalReference = pspReference
};

var captureResult = modification.Cancel(captureRequest);

cancelOrRefund

Similar to cancel, using this you can choose to cancel or refund the payment.

var client = new Client();
var modification = new Modification(client);

var captureRequest = new CancelOrRefundRequest
{
  MerchantAccount = "MerchantAccount",
  Reference = "Merchant reference ",
  OriginalReference = pspReference
};

var captureResult = modification.CancelOrRefund(captureRequest);

refund

With refund you have to specify the amount you want to refund. You can also choose to do a partial refund.

var client = new Client();
var modification = new Modification(client);

var captureRequest = new RefundRequest
{
  MerchantAccount = "MerchantAccount",
  ModificationAmount = new Amount("EUR", 150),
  Reference = "Merchant reference ",
  OriginalReference = pspReference
};

var captureResult = modification.Refund(captureRequest);
